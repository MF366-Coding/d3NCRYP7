## Modes
Current modes and their names:

- **QWERTY Mode** (can be called using ``qwerty_mode``, ``qwerty`` and/or ``mode_qwerty``)

- **Starts from the letter F** (can be called using ``f_start``, ``f_starts``, ``start_f`` and/or ``starts_f``)
